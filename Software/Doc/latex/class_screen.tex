\doxysubsection{Detailed Description}
Classe principale pour gérer l\textquotesingle{}affichage, les boutons et les jauges sur un écran TFT tactile. 

Permet d\textquotesingle{}initialiser l\textquotesingle{}écran, d\textquotesingle{}afficher des images, des jauges, de gérer les menus, les boutons tactiles et la calibration.
