\doxysubsection{Detailed Description}
A real value returned by a function whose ascending root is to be found. 

When calling Astronomy\+\_\+\+Search, the caller must pass in a callback function compatible with the function-\/pointer type astro\+\_\+search\+\_\+func\+\_\+t whose ascending root is to be found. That callback function must return \doxylink{structastro__func__result__t}{astro\+\_\+func\+\_\+result\+\_\+t}. If the function call is successful, it will set {\ttfamily status} to {\ttfamily ASTRO\+\_\+\+SUCCESS} and {\ttfamily value} to the numeric value appropriate for the given date and time. If the call fails for some reason, it should set {\ttfamily status} to an appropriate error value other than {\ttfamily ASTRO\+\_\+\+SUCCESS}; in the error case, to guard against any possible misuse of {\ttfamily value}, it is recommended to set {\ttfamily value} to {\ttfamily NAN}, though this is not strictly necessary.
