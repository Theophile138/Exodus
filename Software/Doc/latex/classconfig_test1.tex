\doxysubsection{Detailed Description}
Classe statique de configuration pour un test matériel avec registre à décalage, multiplexeur, capteurs et actionneur sur ESP32. 

Cette classe regroupe la configuration des broches, l\textquotesingle{}initialisation des objets matériels (registre, multiplexeur, capteurs, moteur) et fournit des méthodes statiques pour initialiser et exécuter le test.
