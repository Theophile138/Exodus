\doxysubsection{Detailed Description}
Classe pour piloter un vérin via deux sorties digitales. 

Cette classe permet de contrôler la montée, la descente et l\textquotesingle{}arrêt d\textquotesingle{}un vérin en utilisant deux sorties digitales, qui peuvent être connectées soit directement à l\textquotesingle{}ESP32, soit à un registre à décalage via l\textquotesingle{}interface \doxylink{class_parent_device}{Parent\+Device}.
