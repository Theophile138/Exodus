Detailed Description

Classe pour piloter un vérin via deux sorties digitales.

Cette classe permet de contrôler la montée, la descente et l'arrêt d'un vérin en utilisant deux sorties digitales, qui peuvent être connectées soit directement à l'ESP32, soit à un registre à décalage via l'interface ParentDevice.
