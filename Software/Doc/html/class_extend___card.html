Detailed Description

Classe permettant de gérer des extensions d'entrées/sorties grâce à un registre et deux multiplexeurs.

Cette classe facilite étend le nombre d'entrées/sorties sur un ESP32 en utilisant un registre pour les sorties digitales et deux multiplexeurs pour les entrées analogiques.
